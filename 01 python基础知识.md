## 运算符
### 算数运算符
- `+`	加
- `-`	减
- `*`	乘
- `/`	除
- `%`	取模
- `**`	幂
- `//`	取整除

### 逻辑运算符
- `x or y` if x is false, then y, else x
- `x and y` if x is false, then x, else y
- `not x` if x is false, then True, else False
> (短路逻辑)

### 比较运算符
- `==` #两对象相等
- `<`
- `>`
- `>=`
- `<=`
- `!=`
- `is` #两对象的同一性
- `is not`
- `in` #成员资格
- `not in`
> 注意`==`和`is`的区别
> 字符串比较：按照ASCII顺序


## 标点符号
- `,` 输出两个语句中添加逗号相当于输出一个空格
- `+` 输出两个语句中添加加号相当于两个语句拼接
- ` ` 输出两个语句中添加空格相当于一个语句
