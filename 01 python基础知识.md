##### python中的下划线`_`
- 单下划线`_`
    - **在解释器中**：执行上一条语句的结果，如:
        ```python
        >>> 42
        >>> _
        42
        ```
    - **作为临时名称**：在语句中作为临时名称使用，当其他人阅读你的代码时将会知道，你分配了一个特定的名称，但是并不会在后面再次用到该名称。例如：
        ```python
        n = 42
        for _ in range(n):
            do_something()
        ```
    - **国际化**
- 名称前的单下划线：私有变量，只供内部使用，除非`__all__`列表显式地包含了它们。

> - 注意：
>   - `_xxx` 不能用`from module import *`导入
>   - `__xxx__` 系统定义名字
>   - `__xxx` 类中的私有变量名

核心风格：避免用下划线作为变量名的开始
